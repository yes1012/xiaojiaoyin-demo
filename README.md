# 小脚印 AI 旅游助手 MVP Demo

小脚印是一款 AI Native 旅行助手概念 Demo，核心理念是“每个人的脚印都是独一无二的，每个人的旅途也是独一无二的”。

这个 Demo 展示了移动端优先的完整体验：

- 创建个性化 AI 旅行搭子
- 生成杭州周末行程
- 查看西湖附近推荐
- 地点打卡并解锁徽章和 BGM
- 记录彩蛋与“小水坑”
- 汇总旅行回忆
- 支持好友、旅游团和 App 内分享

## 在线打开

部署到 GitHub Pages 后，访问仓库的 Pages 地址即可打开：

- App Demo: `index.html`
- 使用说明书: `manual.html`

## 本地预览

直接打开 `index.html` 即可，也可以用本地静态服务预览。

## 文件说明

- `index.html`: 小脚印 App Demo
- `styles.css`: App 视觉样式
- `app.js`: Demo 交互逻辑
- `manual.html`: 网页版使用说明书
- `manual.css`: 使用说明书样式
- `assets/`: Demo 图片资源
- `小脚印Demo使用说明书.md`: Markdown 版说明书
